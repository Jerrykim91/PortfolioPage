God Damn Project.📥

# 목표

  • 장고를 이용하여 데이터 처리 패키지를 사용하지 않고 분석하는 것이 핵심

# 주제

  • 사용자 보고자 하는 년도 나라를 선택 해당 데이터를 그래프화
  • GDP를 보기 쉽게 정리하여 그래프화
  • 선택한 나라의 GDP와 인구 값을 그래프화
  • GDP중 사용자가 필요한 데이터만 json으로 다운로드 가능
  • 크롤링을 이용해 국기, 나라 정보를 보여줌



# 역활

  • 일정관리, 기획
  • 웹서비스 개발
  • 운영 체제 : Windows, Ubuntu(Docker)
